## Setup project

```shell
composer install
```
```shell
npm i
```

create `.env` with you're access to db

```shell
php artisan key:generate
```

```shell
php artisan migrate
```

```shell
php artisan db:seed
```

```shell
php artisan storage:link
```
- ``(!Warning locally in .env) APP_URL=http://127.0.0.1:8000``
